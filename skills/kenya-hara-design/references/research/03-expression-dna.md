# 原研哉（Kenya Hara）视觉表达 DNA 与设计语言碎片

> 研究目标：提炼原研哉的视觉直觉规则——不是他"说了什么"，而是他"怎么做的"。每条规则标注具体案例、时间线和来源。

## 一、视觉签名：反复出现的视觉特征

### 1.1 海报设计

#### 无印良品「地平线」系列（2003，玻利维亚乌尤尼盐湖 + 蒙古草原）

**这是原研哉视觉语言最纯粹的一次释放。** 没有产品包装约束，是他在 MUJI 品牌框架下获得的最大自由度。

| 特征 | 具体描述 |
|------|----------|
| **地平线位置** | 画面正中央，天空与地面/水面各占 50%。区别于经典三分法，原研哉主动拒绝三分法构图。 |
| **人物比例** | 画面中的人物（如一个 14 岁女孩）约占画面高度的 1%-2%，功能是尺度参照物而非主体。 |
| **景深** | 从前景到无限远全部清晰，无虚化。 |
| **颜色** | 高度去饱和。盐湖版近乎单色（白/浅蓝/灰），蒙古版为极淡的黄绿色。 |
| **光线** | 均匀散射光，无强烈光影。避免黄金时刻的戏剧性光效。 |
| **Logo 位置** | MUJI 标志悬浮于地平线上，大小极小，"像远方的巨石阵"。 |
| **文案** | 零文案——无标题、无标语、无产品图、无包装图。 |
| **灵感来源** | 杉本博司《海景》系列——画面正中央水平线分割天空与海，无时间/地点/季节标识。 |

**构图公式可总结为**：中央地平线（天:地=1:1） + 超广角极致景深 + 人物占比 <2% + 均匀散射光 + 零文案 + 极小 Logo 悬浮地平线。

#### EXPO 2005 爱知世博会（1998-2000 创作）

| 特征 | 具体描述 |
|------|----------|
| **主视觉素材** | 江户末期《本草图说》（高木春山绘）的动植物插图，非摄影照片。 |
| **色彩策略** | 异常大胆——朱红色为主色，辅以白底。这是原研哉罕见使用高饱和度颜色的案例。 |
| **构图模式** | 动植物插图居中或偏上，大量留白环绕，世博会标志和文字信息置于底部或边缘。 |
| **设计哲学** | 用江户时期的博物画而非科学摄影，传达"对自然的朴素敬畏"而非"冷冰冰的科学凝视"。 |
| **字体** | 日文汉字和拉丁字母并行，呼应世博会的国际性。 |

**与地平线系列的反差**：地平线系列是"空"的极致——几乎什么都没有；EXPO 海报是"填"的节制——有丰富内容但用白色空间包裹。两条线都展示了"白色作为容器"的核心手法。

#### Hiroshima Appeals 2017 和平海报

| 特征 | 具体描述 |
|------|----------|
| **视角** | 从正下方仰望蘑菇云——这在所有原子弹影像中极为罕见。通常的蘑菇云照片是从侧面水平拍摄的。 |
| **图像处理** | 超写实插画（合作插画师水谷嘉孝），非照片。最终版本裁切后"踏入爆炸内部"——原画包含完整蘑菇顶，最终版裁掉边缘以增强沉浸感。 |
| **情感策略** | 不直接呈现受害者的痛苦（原研哉承认"无法真正再现"），而是通过一个前所未见的视角让观者"凝视那一刻"。 |
| **印刷用纸** | VENT NOUVEAU V 雪白纸，1091 × 788 mm，175 gsm。 |
| **原研哉原话** | "海报有能力将褪色的现实和情感冻结为永恒……就像在创造祈祷。" |

### 1.2 书籍设计

#### 《Designing Design》（2007，Lars Muller Publishers）

| 特征 | 具体描述 |
|------|----------|
| **开本** | 24 × 17 cm 竖版（接近 A5 的细长比例），467 页。 |
| **封面** | 纯白织物硬封，白上加白，无任何图像。织物会随着翻阅留下手印和污迹——这**不是缺陷，是故意的设计**。 |
| **环衬/护封** | 半截白色护封，"建议拆掉以感受织物封面"。 |
| **内页纸** | 多种不同纸材分章节使用，翻页过程即触觉探索。 |
| **印刷工艺** | 压凹、烫金、局部 UV、模切（露出下层图像）、贴页（tip-in）。 |
| **网格系统** | 全书 467 页使用统一的"严谨标准网格"，极度沉静，散发白色带来的严肃感。 |
| **文字排版** | 日文竖排与英文横排共存，在传统和现代之间切换。 |

原研哉称这本书为"信息雕塑"（information sculpture）——信息被雕刻在纸上。

#### 《白》（Shiro, 2010）

| 特征 | 具体描述 |
|------|----------|
| **封面** | 整版纯白，仅左上角黑色书名。 |
| **概念** | "这不是一本关于颜色的书。"白被定位为设计概念而非色素。 |
| **内容结构** | 探讨纸的折叠、字体从白中出现、白沙与月光——将"白"与感官体验而非人工色彩绑定。 |

#### 《Ex-formation》（2015）

480 页，编纂了原研哉在武藏野美术大学十年研讨班学生作品。开本与《Designing Design》一致。

#### 《Draw》（2025）

| 特征 | 具体描述 |
|------|----------|
| **内容** | 352 页，720 幅黑白插图，跨越 40 年（1983-2024）的铅笔草图。 |
| **开本** | 约 15.9 × 24.8 cm（6.25 × 9.75 英寸）平装。 |
| **结构** | 按四个时期分章：1983-2000 / 2000-2010 / 2010-2020 / 2020-至今。 |
| **意义** | 首次公开私人素描本，展示从笨拙初学到自信概念草图的完整轨迹。 |

#### 书籍设计总结规律

- 开本偏好 24 × 17 cm 附近的竖版比例（接近 1:√2 的国际标准纸比例）。
- 封面几乎永远白底，极少图像，书名黑色小字。
- 每本书的材质本身就是该书的论述实体——《Designing Design》讲触觉，书就用织物；《白》讲白色，书就白到底。
- 所有 Lars Muller 出版的英文版遵循统一的极简设计语言。

### 1.3 VI / 品牌设计

#### MUJI（2002-至今）

| 特征 | 描述 |
|------|------|
| **Logo** | 深红色无衬线日文+英文组合标志，比例极小，在广告中始终"悬浮"而非"占据"画面。 |
| **广告哲学** | "空容器"——广告不植入信息，而是创造一个让观者自行填入意义的空间。 |
| **产品色彩** | 米色、灰色、褐色、蓝色——全部低纯度色系。 |
| **包装** | 极简到几乎不存在——透明塑料袋+小标签，或裸装无包装。 |
| **产品哲学** | "这样就好"（これでいい）而非"我想要这个"——对消费主义的反向表态。 |

#### 梅田医院（1998）

| 特征 | 描述 |
|------|------|
| **核心材料** | 可拆卸清洗的白色棉布——所有导视牌覆盖在白色棉布套中。 |
| **颜色** | 白色棉布 + 红色小字（日语）。极简到只有白和红两个颜色。 |
| **触觉策略** | 布料柔软而非硬质塑料/金属板，软化医院氛围。 |
| **信任建构** | "可洗白棉布"类比一流餐厅的白色桌布——既然敢用白色，说明清洁标准极高。 |

#### 茑屋书店（2011）

| 特征 | 描述 |
|------|------|
| **Logo 处理** | 将品牌文字"粒子化"——拆解为点的集合排列（栅格化/像素化），黑白色。 |
| **VI 策略** | 通过粒子化统一汉字和英文字母的视觉语言，形成独特识别度。 |

#### 八海山酿酒（百周年 CI 更新）

| 特征 | 描述 |
|------|------|
| **设计难点** | 八海山已从清酒扩展到烧酒、啤酒、威士忌、金酒，Logo 需在所有瓶型上通用。 |
| **解决方案** | 在统一字形中引入"每种酒类的微妙个性"——字母形态有轻微差异以唤起"亲近感和依恋"。 |
| **团队** | AD: 原研哉, CI设计: 戴永强, 设计: 原研哉/大桥加奈子/星野朝纪。 |

#### 小米 Logo（2021）

| 特征 | 描述 |
|------|------|
| **设计方法** | 用"超椭圆"数学公式 │x│^n + │y│^n = 1, n=3 生成形状，精确处于方形和圆形之间。 |
| **设计理念** | "Alive"——用东方哲学连接人性与技术。 |
| **争议** | 3 年、200 万人民币、仅把方角变圆角——引发全球群嘲。但超椭圆公式的连续曲率与 CSS border-radius 的视觉断点有本质区别。 |
| **原研哉的真正逻辑** | "人们不是被形状打动，而是与品牌背后的理念产生共鸣。" |
| **后续验证** | 小米 SU7 电动车上市后，圆角车标在车身曲面上的效果远好于直角标——超椭圆设计具有前向兼容性。 |

### 1.4 展览空间设计

#### RE-DESIGN（2000，全球巡展）

32 位日本顶级创作者（坂茂、隈研吾、深泽直人等）重新设计日常物品：

- **坂茂的方形卫生纸卷**：四角形芯（非圆形），拉纸时产生阻力和"咔嗒"声，暗示"节约用纸"。储藏和运输也更高效。
- **佐藤雅彦的出入境章**：入境章是飞机"向左飞"，出境章是飞机"向右飞"——用方向暗示行程，不用读字即懂。

#### HAPTIC（2004）

探索五感设计的展览。展品包括：凝胶门把手、仿果皮纹理的果汁盒、盲文状纸面。

#### SENSEWARE / TOKYO FIBER '09（2009）

15 位创造者 + 2 家公司，以日本先进人造纤维为媒材探索"唤醒感官的物体"：
- 斥水织物上用水滴写出文字
- 嵌光纤的水泥砖能"看到"背后的东西
- 迎客自动发光的椅子
- Nendo 的蘑菇形聚酯灯——像吹玻璃一样吹塑而成
- 95% 是空气的 BREATHAIR 积木

**核心理念**："Senseware = 能激发人类创作本能、唤醒制造欲望的物。"

#### HOUSE VISION（2013 东京 / 2016 东京 / 2018 北京）

以"家"为透镜，探索产业的未来。原研哉坚持用 **1:1 实体模型**，让参观者走进去、触碰、"明白"：

- **2013 主题**："以新的常识设计家"
- **2016 主题**："Co-Dividual — 分裂又连接/分离又聚合"（回应日本少子化和独居潮）
- **2018 北京主题**："New Gravity"

**空间设计特点**：
- 消融室内/室外界限（伊东丰雄的土间, 藤本壮介的嵌套框架）
- 高技术 x 传统并存（智能能源 + 杉木工艺）
- 房屋不是产品，是"持续的服务关系"
- 集体奢侈替代私人空间（藤本的租赁空间塔——最小化私有空间，最大化共享空间）

## 二、色彩 DNA

### 2.1 原研哉用色的核心公式

| 元素 | 规则 | 张力案例 |
|------|------|----------|
| **白** | 无处不在。不是"颜色"而是"设计概念"。承载一切。 | 在《白》一书中论证：白 = "全色"且"无色"同时成立。 |
| **黑** | 文字用黑（白底黑字），偶尔背景用黑以凸显白色。 | 黑白对立构成基础张力。 |
| **红** | 极少出现。出现时必为日本国旗红或 MUJI 标志红。小面积、高纯度、白底或棉布底。 | 梅田医院（白色棉布+红色小字）、日本国旗隐喻。 |
| **大地色系** | MUJI 产品线的默认色域：米色、浅棕、灰褐、墨绿、灰蓝。全部低饱和度。 | 这是 MUJI 品牌约束色系，非原研哉个人自由选择。 |
| **高饱和色** | 在他完全自主的项目中几乎不存在。唯一例外：EXPO 2005 使用了朱红色主调。 | 朱红来自江户传统，非原研哉个人偏好——是一次"为命题服务的例外"。 |

### 2.2 白的使用分类

| 白的类型 | 作品案例 |
|----------|----------|
| **概念白** | 《白》全书——白作为设计哲学而非颜色。 |
| **空间白** | 地平线海报——70-80% 画面是白色天空或浅色地面。 |
| **材质白** | 梅田医院棉布、《Designing Design》白织物封面。 |
| **纸白** | 所有书籍封面——白色纸张作为"开始的空白"。 |
| **过程白** | 原研哉谈"清洁/维护白色"作为一种审美行为——花园之美在持续擦拭后才完全显现。 |
| **100 种白** | 2019 年出版《100 Whites》——在看似相同的白色中区分 100 种细微差异。 |

### 2.3 色彩饱和度范围

量化归纳：

| 语境 | 典型饱和度 |
|------|-----------|
| **MUJI 产品** | 0-15%（几乎不可感知的色相） |
| **MUJI 广告** | 10-30%（自然景观的去饱和版本） |
| **个人作品（海报/VI）** | 0-20%（除非有特定命题要求如 EXPO） |
| **书籍封面** | 0%（纯白）+ 100%（纯黑文字） |
| **梅田医院** | 0%（白）+ 100%（一点红色） |

### 2.4 原研哉对色彩的根本立场

> "工业将色彩从原本的载体（白云、蓝天、绿叶）中剥离，变成编码合成的标准化产品，失去了传递真正情绪的能力。还原色彩的本质，就是让色彩回归肌理载体之中。"

**翻译成设计直觉规则**：选择颜色时，永远问"这个颜色附着在什么物质上？"而非"这个颜色本身好不好看？"。CMYK 值在你没有确定纸张/材质之前没有意义。

## 三、排版习惯

### 3.1 字体选择

| 场景 | 推测字体/倾向 | 依据 |
|------|-------------|------|
| **日文正文/标题** | 明朝体（Mincho，有衬线日文字体）为主，字体极细、字间距宽松。 | 所有 MUJI 广告和 Hara Design Institute 作品呈现出极其纤细、优雅的明朝体特征。 |
| **拉丁字母** | 无衬线体（类似 Helvetica / Frutiger 系列），极细字重（Light/Thin）。 | MUJI 品牌标志和相关英文排版。 |
| **书籍正文** | 日文竖排用明朝体；英文/数字横排用无衬线体。 | 《Designing Design》内页排版。 |
| **特殊处理** | 茑屋书店：粒子化文字（栅格化，点阵构成）。八海山：字形的微妙个性化差异。 | 品牌 VI 项目。 |

**核心规律**：原研哉不迷恋特定字体名称。他关注的是字体在特定材质上的"存在感"——字体的粗细、间距、大小与留白的关系。同一字体印在涂布纸上和印在棉布上的"重量"完全不同。

### 3.2 字间距与行距

| 参数 | 模式 |
|------|------|
| **字间距** | 宽松。日文字距通常比日本常规排版更松。英文字距趋向极宽（tracking +50~+150）。 |
| **行距** | 宽松。日文行距通常为正文字号的 1.8-2.5 倍。 |
| **文字在版面中的占比** | 极低。文字通常只占版面的 5-15%，其余为白色空间。 |
| **字号对比** | 标题和正文之间字号反差极小——不靠字号吼叫，靠空间区分层级。 |

### 3.3 文字与空白的关系

原研哉的排版核心公式：

> **文字不是印在"背景白"上，文字是"白色空间中出现的少量黑色痕迹"。白色是主动的、有质量的、有含义的空间——不是默认的空背景。**

具体表现：
- 《白》封面：整版白（约 95%），书名黑色小字在左上角（约 5%）。
- 地平线海报：MUJI 标志约占总面积 <1%。
- "无何有"旅馆案例：中间的"何"字笔画被刻意省略，使其既像"无"又像"有"——阅读过程中体验"从无到有"。

### 3.4 日文竖排 + 英文横排的并行

长野冬奥会闭幕式节目册（1998）：日语竖排 + 法语/英语横排——阅读方向的交错创造了一种动态的视觉韵律。这个手法在后续书籍设计中反复出现。

## 四、图像处理方式

### 4.1 摄影处理

| 特征 | 描述 |
|------|------|
| **对比度** | 低对比度优先。地平线系列使用均匀散射光，避免高光/深阴影。 |
| **色调** | 冷调或中性调。极少暖调（除非自然场景本身暖如蒙古草原）。 |
| **饱和度** | 去饱和处理。MUJI 广告中的自然景观都被"抽掉"了一部分色彩。 |
| **黑白** | 《Draw》全书中 720 幅全是黑白。他的个人视觉思维本身是单色的。 |
| **清晰度** | 极致清晰——地平线系列从前景到无限远无一处虚化。 |

### 4.2 图像在版面中的比例和位置

| 模式 | 案例 |
|------|------|
| **图像居中、满幅到边缘** | 地平线系列——图像填充整个画面，边距为零。 |
| **图像居中、被大量白边环绕** | EXPO 2005 海报——动植物插画居中，周围宽白边。 |
| **图像极小、漂浮在白色空间中** | 《白》杂志封面系列——白色物体几乎消失在白背景中（white-on-white）。 |
| **图像被裁切成特写** | Hiroshima Appeals 2017——裁掉蘑菇云的边缘，"踏入爆炸内部"。 |

### 4.3 图像处理总结

原研哉处理图像的方式可以归纳为两个极端：
1. **要么让图像吞没整个画面**（地平线——无边无际）——此时白被"内置"在图像中（白色天空/地面）。
2. **要么把图像关进白色笼子里**（EXPO、书籍封面）——此时白是"外框"，图像是框中陈列物。

中间地带几乎不存在。

### 5.4 触觉哲学

原研哉的材质选择有一条底层逻辑链：

- **触觉是五感之首** → 材质决定触觉 → 触觉唤起记忆 → 记忆构成感知 → 感知即设计。
- 他明确提出"把触觉通过视觉传达出来"——好的平面设计应该让观者"看到"材质的触感。柳宗理的蝴蝶凳纹理、纸纤维的绒面、棉布的柔软——都必须在印刷平面上被"感受到"。

> "Haptic = 不在乎如何创造，而在乎如何让人感知到。" — 原研哉

## 六、构图规则：可总结的模式

### 6.1 地平线模式（对称 + 均衡）

| 参数 | 值 |
|------|-----|
| 地平线位置 | 画面正中央（50%高度处） |
| 天空:地面比例 | 1:1 |
| 人物大小 | 画面高度的 1-2% |
| 文字/Logo | 极小，悬浮于地平线上 |
| 景深 | 全焦（f/11-f/16 级别） |
| 适用场景 | MUJI 广告、品牌形象海报 |
| 代表作 | 玻利维亚盐湖、蒙古草原 |

### 6.2 白底居中模式（静态 + 孤立）

| 参数 | 值 |
|------|-----|
| 主体位置 | 画面几何中心 |
| 主体占比 | 20-40%（非满幅） |
| 白边比例 | 60-80% |
| 文字 | 底部或顶部边缘，极简 |
| 适用场景 | EXPO 海报、展览海报 |
| 代表作 | EXPO 2005 Aichi 系列 |

### 6.3 满幅图像模式（零边距 + 沉浸）

| 参数 | 值 |
|------|-----|
| 图像 | 填充全部版面，无边距 |
| 白不是外框而是图像内容的一部分 | 图像内部的浅色/白色区域充当"内置留白" |
| 文字 | 叠加在图像上，极简极淡 |
| 适用场景 | MUJI 广告系列 |
| 代表作 | 地平线、冰原、加拉帕戈斯 |

### 6.4 白色物体漂浮在白色空间中（white-on-white）

| 参数 | 值 |
|------|-----|
| 背景 | 纯白（纸白） |
| 物体 | 纯白或极浅色物体，靠微妙阴影或纹理辨识 |
| 可见性 | 极低——需要靠近、暂停、仔细看 |
| 功能 | 唤醒注意力、制造"发现"的快感 |
| 代表作 | 《白》杂志封面系列、竹尾纸业海报 |

### 6.5 被裁切的沉浸构图（后期出现，2017 后）

| 参数 | 值 |
|------|-----|
| 图像裁切 | 故意裁掉边缘，"踏进画面内部" |
| 功能 | 增强紧迫感、打破旁观者视角 |
| 代表作 | Hiroshima Appeals 2017 |

### 6.6 整体规律总结

| 维度 | 原研哉的倾向 |
|------|------------|
| **对称 vs 不对称** | 偏向绝对对称（地平线中央分割）或中心对称（主体居中）。极少使用对角线/动态不对称。 |
| **静态 vs 动态** | 极强偏向静态。他的画面"不运动"——没有视觉引导线、没有紧张的对角线、没有戏剧性的方向力。 |
| **满幅 vs 留白** | 两个极端都擅长：要么零留白（满幅图像），要么 60-80% 留白。从不做 50-50 的中间状态。 |
| **文字位置** | 边缘或悬浮——永远不"控制"画面中心。 |
| **装饰元素数量** | 零。连分割线、色块、图标都极少出现。 |

## 八、争议性设计立场（"反主流"观点）

### 8.1 "无设计"（No Design）——看似取消设计，实则取消的是设计师的自我表达

> "无设计"不是不设计，而是让设计师的意图尽可能不可见——创造出感觉'自然'而非'被创作出来'的物品。"

**反主流之处**：在设计师被鼓励建立个人风格、签名式视觉语言的时代，原研哉主张设计师应该匿名化自己。产品不应该让人想到"这是某个厉害设计师的作品"，而应该让人想到"这东西本来就该是这个样子。"

### 8.2 "这样就好" vs "我想要这个" —— 直接对抗消费主义

> "MUJI 不做'我想要这个'的产品，只做'这样就好'的产品。"

**反主流之处**：消费主义的核心驱动力是欲望和占有。原研哉用"满足"替代"欲望"，用"足够"替代"更多"。在品牌=制造欲望的行业里，MUJI 是唯一一个把"不制造欲望"当作品牌主张的商业品牌。

### 8.3 反对数据驱动设计

> "人们问 AI 可以做设计吗？我说，请先理解设计的本质是什么。信息太多会阻碍好的设计。"

**反主流之处**：在数据驱动、A/B 测试、用户画像主导设计的时代，原研哉主张"Un-knowing"（ex-formation）——把已知变成未知，重新发现好奇心。他认为设计不是解决问题的工具，而是"发现真正问题"的过程。

### 8.4 不靠视觉冲击力做沟通

> "沟通的理想不是用抓人眼球的画面来捕捉注意力，而是让画面渗透入五感。这种沟通非常难以捉摸但极其坚固，在我们还没意识到它存在之前就已经成功了。"

**反主流之处**：主流通用设计教育强调"视觉冲击力"、"第一眼抓住用户"、"信息层级"。原研哉反其道而行——好设计应该在你没注意到时就完成了沟通。这是一种"隐形的说服"。

### 8.5 对"简洁"的重新定义

> "西方所说的'简洁'（simplicity）类似于日本的'空'（emptiness）。但 MUJI 的简洁应该'华丽而不带自卑感'。"

**反主流之处**：极简主义常被批评为"冷感"、"疏离"。原研哉直面这个批评，并给出一个悖论式的回应——MUJI 的简洁不是寒酸的、自卑的、删减后的匮乏。恰恰相反，它是丰盛的、华丽的——只是这种华丽不是用装饰来表达的。

### 8.6 "Design begins with comprehension of the unknown"

> "设计始于对未知的理解。'我知道我知道'是最糟糕的概念——这意味着你的作品没有任何新东西。要成功，必须自觉地寻找未知。"

**反主流之处**：大多数设计方法论从"理解用户/问题/市场"开始——即从"已知"出发。原研哉把这个起点完全反转：从"未知"出发，从"我其实什么都不懂"出发。

### 8.7 "我们不做广告"（MUJI 地平线系列的实际效果）

尽管地平线海报在形式上是广告，但零产品、零文案、零促销——原研哉实质上取消了广告的所有商业元素，却创造了品牌广告史上最成功的案例之一。

## 九、个人风格 vs MUJI 品牌约束

### 9.1 两种模式的区分

| 维度 | 原研哉个人风格（无品牌约束） | 原研哉在 MUJI 框架内（品牌约束） |
|------|--------------------------|-------------------------------|
| **色彩自由度** | 可用朱红（EXPO）、金箔（竹尾纸业）、黑+白 | 严格限制：大地色系 + 白 + 黑 + 一点红 |
| **图像来源** | 插画（EXPO 江户博物画）、写实插画（Hiroshima）、抽象纹理（竹尾纸业） | 几乎只用摄影（自然景观） |
| **构图的实验性** | 高——粒子化文字、white-on-white 极限、裁切闯入 | 稳定在"地平线模式"和"满幅自然"两种 |
| **触觉实验** | 极端——棉布导视、果皮纹理、金箔 | 温和——产品材质本身（棉、木、纸） |
| **哲学表达** | 直接输出——展览、著作、教学 | 通过品牌过滤——"空容器的哲学"包装为"无品牌哲学" |
| **最纯释放** | 《白》书籍、竹尾纸业海报、Hiroshima Appeals | 地平线系列（MUJI 给他的最大自由度时刻） |

### 9.2 关键判断

- **地平线系列虽然署名 MUJI，实际上是原研哉个人视觉 DNA 最纯粹的一次商业释放。** MUJI 的品牌约束（无产品、无促销文案、无色彩）在此刻不再构成约束，而是变成了原研哉哲学的商业载体。
- **竹尾纸业海报（1989-1994）是他加入 MUJI 之前的最纯个人表达**——抽象形式 + 金箔/果皮纹理 + white-on-white 实验。这些实验性手法在 MUJI 版本中被"净化"为更干净、更克制、但也少了实验野心的商业版本。
- **MUJI 约束不是限制，是催化剂**。MUJI 给了原研哉一个必须在极窄色域内发声的挑战——结果是把"克制"本身发展成了一套语言系统。个人项目中偶尔出现的色彩（EXPO 朱红色）恰好反证了 MUJI 框架如何彻底约束了他。

## 十、设计直觉规则清单（供 skill 构建参考）

以下是从所有研究材料中提取的、可程序化执行的规则：

### 10.1 色彩规则
1. 白不是背景，是设计元素。每次使用白色，定义"这是什么类型的白"（概念白/空间白/材质白/纸白/过程白）。
2. 永远问"这个颜色附着在什么材质上？"——脱离载体的颜色不存在。
3. 饱和度控制在 0-20% 范围内。超过 20% 需要特殊的文化/命题理由（如日本红）。
4. 黑只用于文字，不用作大面积色块。红只用于小面积强调（Logo、关键标注）。

### 10.2 构图规则
5. 优先中央对称/中心对称。避免对角线动态构图。
6. 画面有两种有效状态：(a) 物体极小 + 大量留白 (60-80%) 或 (b) 图像满幅 + 白内置在图像内容中。不做 50-50。
7. 如果必须在满幅图像中放入文字/Logo，使其"悬浮"而非"占据"——字号为整体画面的 <2%。
8. 地平线放正中央（50%），不做三分法。

### 10.3 排版规则
9. 字间距放宽松——英文 tracking +50~+150。行距为字号的 1.8-2.5 倍。
10. 文字在版面中占比不超过 15%。层级靠空间区分，不靠字号吼叫。
11. 日文用明朝体（有衬线），极细字重。英文用无衬线 Light/Thin。
12. 封面永远"白底+黑色小字"——不靠图像吸引。

### 10.4 材质规则
13. 设计一个印刷品前，先选定纸张/材质——材质决定一切后续决策。
14. 能触摸的地方，材质必须说话。不能触摸的印刷品，用视觉唤起触觉联想。
15. 故意允许材质"老化"——一本书被翻阅留下痕迹不是缺陷，是使用证据。

### 10.5 图像规则
16. 摄影图像保持低对比度、均匀光线、去饱和 → 给人"自然/永恒"感而非"戏剧/瞬间"感。
17. 图像裁切——考虑是否"踏入画面内部"来增强沉浸感（后期手法，慎用）。
18. 人物的功能是尺度参照，不是叙事主体。如果必须出现人，让他占比 <2%。

### 10.6 创意方法规则
19. 开始项目前，先问"我对这个主题真正知道什么？"——然后推翻自己的答案（ex-formation）。
20. "问更好的问题，而非找更好的答案"——当设计卡住时，说明你问错了问题。
21. 设计不是为了被注意。好设计在你注意到它之前就已经完成了沟通。
22. 永远保留铅笔和纸作为核心工具。数字化之前，先在纸上"释放灵魂中的图像"。
